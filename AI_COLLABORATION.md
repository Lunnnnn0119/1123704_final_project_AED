# AI 協作紀錄摘要

本專題使用 AI 作為程式開發輔助工具，協助範圍如下：

1. 協助建立 Windows 的 Python 虛擬環境、安裝 PyTorch 與 Ultralytics YOLO 的指令。
2. 協助解釋 YOLOv8 資料集結構，並產生訓練、評估、影片推論的初版程式。
3. 協助分析執行環境與錯誤訊息，例如 PowerShell 與 CMD 指令差異。
4. 協助整理本 repository 的安裝、測試、開源專案來源與授權說明。

提交前應由專題作者自行執行、檢查與理解下列項目：資料集標註品質、訓練參數、模型評估結果、攝影機推論效果、程式設計邏輯，以及本檔內容。AI 產出僅作為輔助，最終設計、測試、修改與課程報告的責任由專題作者承擔。
