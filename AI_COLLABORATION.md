# AI 協作紀錄

本專題有使用 AI 工具協助整理程式與文件，主要協作內容如下：

1. 協助建立 YOLOv8 訓練環境與安裝流程。
2. 協助撰寫 `train.py`、`evaluate.py`、`infer_video.py`。
3. 協助整理 Windows 可執行的批次檔，例如 `setup.bat`、`run_infer_video.bat`。
4. 協助整理 README、第三方套件說明與 GitHub 專案結構。
5. 協助檢查模型權重、資料集 zip、demo 影片是否正確放入 GitHub。

AI 主要用於程式撰寫輔助、錯誤排除與文件整理；資料蒐集、模型訓練、結果判讀與專題內容仍由作者負責確認。
