# AI 協作紀錄

本專題有使用 AI 工具協助建立 YOLOv8 訓練環境與撰寫部分程式。AI 主要作為程式開發與除錯輔助，協作內容如下：

1. 協助規劃 Windows 上的 Python 虛擬環境與套件安裝流程。
2. 協助設定 Ultralytics YOLOv8 所需的訓練環境。
3. 協助撰寫 `train.py`，用於載入 YOLOv8 模型並訓練 AED 偵測模型。
4. 協助撰寫 `evaluate.py`，用於使用 test split 評估訓練好的模型。
5. 協助撰寫 `infer_video.py`，讓模型可以對影片或攝影機畫面進行 AED 推論。
6. 協助撰寫 `setup.bat` 與 `run_infer_video.bat`，方便在 Windows 環境中安裝套件與執行推論。
7. 協助排除 PowerShell / CMD 執行指令、套件安裝與 YOLO 推論時遇到的錯誤。

AI 協助的範圍主要集中在環境建置、程式撰寫與除錯；資料集內容、模型訓練結果與專題成果由作者自行確認與整理。
